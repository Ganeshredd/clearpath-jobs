Step 01

Save your base resume

Go to My Resume, paste your full resume once, add your LinkedIn URL and save. The optimizer always pulls from this — you never paste it again.

  • Use plain text, not PDF
  • Include all sections
  • Add LinkedIn URL separately
Step 02

Paste the full JD

In the Optimizer, paste the complete job description — including the company overview and benefits. Hidden keywords live everywhere.

  • Include "What You'll Do"
  • Include preferred qualifications
  • More text = better matching
Step 03

Review keyword analysis

Green = already in your resume. Cyan = honestly added from your real experience. Amber = true gaps to address in interviews.

  • 90–100 = Strong match
  • 75–89 = Good, minor gaps
  • Below 75 = Needs work
Step 04

Download and submit

Hit Download DOCX for a perfectly formatted Word document. ATS-safe layout, navy headers, justified text, 8 bullets per role.

  • Works with Workday, Greenhouse, Taleo
  • LinkedIn is a live hyperlink
  • Filename: LastName_JobTitle.docx
Step 05

Save every JD to vault

After optimizing, save the JD. Before every interview, open the vault, find the company, and review the full JD to recall what they want.

  • Add prep notes per job
  • Update status as you progress
  • Search by keyword
Step 06

Verify all numbers

The AI uses realistic benchmarked numbers. Verify each metric before submitting — you must defend every number in an interview.

  • Check asset counts vs actual scope
  • Verify % improvements
  • Match time savings to reality
Pro tip: Run it fresh for every application

No two JDs are identical — even for the same title at different companies. Takes 30 seconds and meaningfully improves your pass rate every time.
